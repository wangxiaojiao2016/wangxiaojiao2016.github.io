关于HTML5和CSS3

  • 继承
    • 字体样式可继承，宽高边框不继承
  • 网站优化
    • 文件合并和压缩，CDN托管和缓存
  • 通信
    • 本地存储来使用，页面之间来通信；WebSocket、SharedWorker，实现通信没问题；Safari无痕模式下，设置localstorage有异常（QuotaExceededError）;websocket兼容浏览器，兼容方法有四种，Adobe Flash Socket、ActiveXHTMLFile(IE)、基于multipart发XHR、基于长轮询XHR
  • HTML5新增标签
    • 语义标签有九个，section、header和footer，nav、article以及main、figure图片、aside，图片注释figcaption；
    • 功能标签有两个，datalist下拉框，下拉列表detail，summary中放detail，detail（open）为打开，默认时候是关闭；
    • 表单种类十二个，search、Tel、email；number、time和地址；range数值选择器；week、month、datetime；
    • 表单特性有八个，聚焦（autofocus）正则（pattern）和关联（form）；占位（placeholder）保存（autocomplete）和必填（required）；on是默认来保存，off取消默认不保存；novalidate不提交，针对form来使用；formnovalidate作用submit上，不验证来提交；表单验证invalid，验证失败来触发
  • 离线存储
    • H5离线存储有原理，基于新建文件的机制（.appcache）；解析清单来离线（资源），资源存储像cookie；离线之后浏览器，页面展示用存储。html头部有manifest,首次访问需存储。
  • iframe的缺点
    • iframe有缺点，1是阻塞主页Onload；2是搜索引擎有问题，无法解锁此页面，不利SEO；页面共享连接池，浏览器对同域有限制，导致并行加载有问题；如果必须要使用，js动态填属性，属性名为src
  • CS3的新特性
    • 圆角多列（multi-column layout）和旋转（transform），阴影（shadow）反射（Reflect）和渐变（gradient）。Cs3页面新布局，Flexbox弹性盒，伸缩容器（flex container）解复杂；子元素是容器，容器元素是项目；常规布局块和（内联）流，flex基于flex-flow流，方便居中自适应，布局空间更灵活。
  • 定位元素position，fixed针对页面定屏幕滑动viewport，Android定位加meta（meta name=”viewport” content=”width=device-width,initial-scale=1.0,maximum-scale=1.0,minimum-scale=1.0,user-scalable=no”>）
  • 静态文件请求
    • 静文请求有cookie（主域名下的静态文件），浪费流量需隔离；cookie不能跨域交请求，（非主要域名时）请求头中没cookie；降低请求大小和时间，减少请求延时性；不传cookie少处理，提高（webserver）解析http的速度
  • CSS处理器
    • LESS、Sass、Stylus，三者均是预处理，增强代码复用性，拥有变量函数等，具有UI组件模块化开发能力更方便，工作效率会更高；PostCSS后处理，规范处理CSS，针对浏览器加前缀，实现跨域浏览器的兼容
  • DOM元素的隐藏
    • display设为none，不显示不占位；visibility为hidden，虽不显示但占位；H5隐藏更简单，标签上面加hidden(\
