Node~Global

  • Global

    • 服务器端没有window对象，后台最大的是global
    • global属性：
      • 在文件中运行 this不是global
      • process 进程对象
        • nextTick 下一队列（当前队列的底部）
        • process.pid 当前进程id
        • process.kill（） 杀死进程
        • process.cwd（） 当前工作目录（可更改）
        • process.exit（） 退出进程
      • Buffer 缓存 16进制
      • clearImmediate :[function] 清除立即
      • setImmediate :[function] 设置立即
      • setTimeOut/clearTimeout
      • setInterval/clearInterval
      • console:[Getter] 输出
      • global上的属性可以在任意地方访问
    • global变量的声明
      • 通过var声明不能直接声明在global上
      • 声明在global上–》global.xxx
      • const 常量
      • 需要变的值–》let

    • console

      • console.log()
      • console.dir()
      • console.info()
      • console.error()
      • console.time(‘start’) 传入的参数（字符串）保持一致
      • console.timeEnd(‘start’) 传入的参数（字符串）保持一致

    • setTimeOut

      • 异步（process.nextTick ;setTimeOut ;setImmediate）

    • 五个全局属性

      • 绝对路径不可变
      • console.log(__dirname) //文件名，不会更改（运行文件的过程中，会给文件套一层闭包函数，函数中要有形参
      • console.log(__filename);//文件模块，
