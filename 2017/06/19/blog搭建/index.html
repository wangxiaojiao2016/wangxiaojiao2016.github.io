blog搭建

  • 1.安装：npm install hexo-cli -g //提供命令 hexo
  • 2.新建文件夹 打开命令窗口
  • 3.初始化：hexo init初始化（默认下载）–》初始化结束后会出现很多文件夹 soure文件下 -post下写代码
  • 4.启动服务：hexo serve 开启博客（Start ）
    • 根据localhost进入 一般是4000端口
    • 端口被占用：hexo serve -p 333 改端口
  • 主题修改（themes文件夹下）-》点进去-》github拷贝地址（git clone+地址）-》断掉服务-》cd 的themes中进行安装 –》config.yml下输入theme：hexo-theme-next–》命令行hexo serve-》端口号窗口查看即可
  • 5.（命令窗口）hexo g 生成发布的内容文件
  • 6.安装插件： npm install hexo-deployer-git –save命令行
  • 7.hexo g
  • 8.github新建仓库 new resposity（名字.github.io 名字格式固定不可修改） create resposity
    • git发布 —config.yml中粘贴发布（注意更改成自己的地址）-》打开命令行 配置用户路径http://userName：password@github.com branch:master
  • 9.hexo deploy 发布 生成了.deploy_git 文件
  • 10.名字.github.io就可以访问了
